Phase 1 — Foundation

Locales Translation Keys
locales/en.default.json

Translation keys power all text labels in the customizer. Every setting label, option name and UI text lives here.

  • Add all general, labels and options keys
  • Add cart, search, password, 404 text strings
  • Make sure t: keys match settings_schema.json
locales/en.default.json
Settings Schema Customizer
config/settings_schema.json

Defines every setting available in Theme Settings — colors, fonts, spacing, buttons, nav and footer. Every id here becomes a settings.id variable.

  • Add typography settings (primary + secondary font)
  • Add color settings with Elunara brand defaults
  • Add button, navigation and footer settings
  • Set brand colors: #FAF3E1 #222222 #FA8112 #F5E7C6
config/settings_schema.json
CSS Variables Design System
snippets/css-variables.liquid

Bridges the customizer settings to CSS. Outputs all values as :root CSS variables including spacing scale and typography scale.

  • Map all settings to --color- variables
  • Add spacing scale: --space-xs through --space-2xl
  • Add typography scale: --text-sm through --text-5xl
  • Load both primary and secondary fonts
snippets/css-variables.liquid
